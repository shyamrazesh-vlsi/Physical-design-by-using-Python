# Universal RPT Analyzer

Professional EDA tool for analyzing Innovus reports (Timing, Power, Area, Congestion, etc.)

## Features

- Universal parser for all Innovus report types
- Automatic report type detection
- Professional dark theme GUI
- Excel export with formatting
- Data visualization
- Multi-file comparison
- Logging system

## Installation

```bash
# Create virtual environment
python -m venv venv
venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Run application
python main.py
